# 今回PoCの標準仕様抽出範囲

## 目的

標準仕様書第6.0版のうち、今回のPoC実装に必要な範囲だけを明確にする。

## 対象業務

| 業務フローID | 名称 | PoCでの扱い |
|---|---|---|
| FLOW-02-01 | 住民情報異動等に伴う資格異動 | Must |
| FLOW-02-02 | 被保険者証等再交付 | Must |

## 対象機能ID

| 機能ID | 名称/要約 | PoCでの扱い |
|---|---|---|
| 0230265 | 住民異動情報をもとにした資格異動 | Must |
| 0230267 | 資格情報照会 | Must |
| 0230268 | 資格情報修正・回復・取消・変更 | Must |
| 0230270 | 65歳到達等の資格異動 | Must |
| 0230273 | 資格情報・異動履歴の一覧確認 | Must |
| 0230275 | 被保険者番号自動付番 | Must |
| 0230286 | 介護保険被保険者証出力 | Must |
| 0230290 | 証発行履歴管理 | Must |
| 0230295 | 再交付・返還状況管理 | Must |
| 0230298 | 被保険者証等再発行 | Must |
| 0230297 | 被保険者証等再交付申請書出力 | Stretch |

## 対象帳票ID

| 帳票ID | 名称 | PoCでの扱い |
|---|---|---|
| 0230010 | 介護保険被保険者証 | Must |
| 0230014 | 介護保険 被保険者証等再交付申請書 | Stretch |

## 対象外

標準仕様に存在しても今回実装しない。

- 住所地特例
- 適用除外施設
- 認定管理
- 給付管理
- 保険料賦課
- 保険料収納
- 滞納管理
- 総合事業
- 国保連連携
- マイナポータル連携
- 本物の住民記録システム連携
- 本番認証/権限/監査/非機能要件
