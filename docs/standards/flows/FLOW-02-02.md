# FLOW-02-02 被保険者証等再交付

## 原典
- 資料: 別紙1 業務フロー 第6.0版
- 大分類: 02 被保険者資格
- 小分類: 02 被保険者証等再交付

## 標準仕様の要約
被保険者証等を紛失・破損・汚損した場合などに、再交付申請を受け付け、対象者を確認し、必要に応じて被保険者証等を作成・送付する流れ。

## 初心者向け説明
住民が「介護保険被保険者証をなくした」「破れてしまった」などの場合、自治体は再交付申請を受け付ける。申請内容と対象者を確認し、新しい被保険者証を再発行する。古い証が返ってきた場合は返還状況も管理する。

## 今回のPoCで扱う流れ
1. 被保険者詳細から再交付申請画面を開く
2. 申請理由を入力する
3. reissue_applications に保存する
4. 再発行実行により certificate_issue_histories に再交付履歴を作成する
5. 被保険者詳細で再交付履歴を確認する

## 今回扱う申請理由
- 紛失・消失
- 破損・汚損
- その他

## 今回扱わない範囲
- マイナポータルぴったりサービス経由のオンライン申請
- 住民記録担当課や施設からの詳細連携
- 返還督促
- 全証種別の再交付
- 交付申請書PDFのMust実装（Stretch扱い）

## 実装上の注意
- 再発行前に reissue_applications が存在すること
- 再発行後は certificate_issue_histories に履歴を残すこと
- care_application_in_progress=true の場合は被保険者証再発行を禁止すること
