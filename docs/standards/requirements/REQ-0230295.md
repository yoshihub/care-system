# REQ-0230295 再交付・返還状況管理

## 標準仕様の要約
被保険者証等の再交付申請や返還状況を管理できること。

## 初心者向け説明
被保険者証を紛失・破損した場合、再交付申請を受け付け、必要に応じて再発行する。古い証の返還状況も管理対象になる。

## 今回のPoCで実装する範囲
reissue_applications に申請情報を保存し、再発行時に certificate_issue_histories を作成する。

## 今回実装しない範囲
返還督促、複雑な申請者管理、全証種別の再交付管理。

- 仕様判断に迷う場合はTODOコメントに残し、勝手に制度解釈を広げないこと
