# REQ-0230265 住民異動情報をもとにした資格異動

## 原典
- 資料: 別紙2 機能・帳票要件 第6.0版
- 機能ID: 0230265
- 関連業務: 被保険者資格
- 関連業務フロー: FLOW-02-01 / FLOW-02-02 の該当箇所

## 標準仕様の要約
住民記録側で発生する住民異動情報をもとに、介護保険の資格取得・変更・喪失につながる情報を管理できること。

## 初心者向け説明
住民が65歳になった、転入した、転出した、死亡した、住所や氏名が変わった、という情報は住民記録システム側で発生する。介護保険システムは、その情報を受け取って、介護保険の資格状態を変える必要がある。

## 今回のPoCで実装する範囲
PoCでは本物の住民記録システムとは連携せず、CSV取込または手入力で resident_change_events に登録する。資格履歴や被保険者の作成は後続タスクで行う。

## 今回実装しない範囲
実住民記録連携、自動バッチ、住所地特例、適用除外施設、住登外者の詳細管理。

## 実装上の制約
- このREQに書かれていない機能を勝手に追加しないこと
- 標準仕様上の対象が広くても、PoCでは「今回のPoCで実装する範囲」に限定すること
- 仕様判断に迷う場合はTODOコメントに残し、勝手に制度解釈を広げないこと
