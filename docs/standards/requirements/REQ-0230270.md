# REQ-0230270 65歳到達等の資格異動

## 原典
- 資料: 別紙2 機能・帳票要件 第6.0版
- 機能ID: 0230270
- 関連業務: 被保険者資格
- 関連業務フロー: FLOW-02-01 / FLOW-02-02 の該当箇所

## 標準仕様の要約
65歳到達者等について、資格取得につながる異動を管理できること。

## 初心者向け説明
介護保険の第1号被保険者は原則65歳以上であるため、65歳到達は介護保険資格管理の重要な入口になる。

## 今回のPoCで実装する範囲
resident_change_events.event_type=AGE_65 を資格取得イベントとして扱い、資格登録時に insured_persons と qualification_histories を作成する。

## 今回実装しない範囲
年齢到達予定者の自動抽出バッチ、事前通知の完全実装。

- このREQに書かれていない機能を勝手に追加しないこと

- 仕様判断に迷う場合はTODOコメントに残し、勝手に制度解釈を広げないこと
