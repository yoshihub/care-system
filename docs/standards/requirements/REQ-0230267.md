# REQ-0230267 資格情報照会

## 標準仕様の要約
被保険者の資格情報を照会できること。

## 初心者向け説明
介護保険担当者が、対象者の被保険者番号、氏名、生年月日、住所、資格状態、資格取得日などを確認できる必要がある。

## 今回のPoCで実装する範囲
被保険者一覧APIと被保険者詳細APIで、基本情報・資格履歴・証発行履歴・再交付申請履歴を確認できるようにする。

## 今回実装しない範囲
高度な検索条件、EUC汎用出力、住所地特例や適用除外施設の詳細情報表示。
