## 正常系 シナリオA
1. 65歳到達の住民異動イベントを登録
2. 候補者一覧に表示
3. 資格取得登録
4. 被保険者番号が付与
5. 被保険者証をプレビュー
6. PDF出力
7. 発行履歴に1件追加

## 正常系 シナリオB
1. 既存被保険者を開く
2. 再交付申請を登録
3. 再発行
4. 発行履歴に reissue が追加

## 異常系
- 必須項目未入力で登録不可
- residentNoが空なら住民異動登録不可
- qualificationChangedAtが空なら資格異動登録不可
- 存在しないinsuredPersonIdでは帳票出力不可
- 重複被保険者番号が作成されない

## デモ台本 5分
1. 「標準仕様書をもとに、被保険者資格の一部をPoC化しました」と説明
2. 住民異動イベント登録
3. 被保険者一覧で候補者表示
4. 資格異動登録
5. 被保険者詳細で履歴確認
6. 被保険者証プレビュー
7. PDFダウンロード
8. 発行履歴確認
9. 余裕があれば再交付申請・再発行
