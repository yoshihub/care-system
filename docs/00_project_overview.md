# 介護標準化システム PoC デモ概要

## 目的
介護保険システム標準仕様書 第6.0版をもとに、AI駆動開発で「被保険者資格」の一部業務を、画面・API・DB・帳票まで一貫して試作する。

## デモ対象
- 02-01 住民情報異動等に伴う資格異動
- 02-02 被保険者証等再交付

## 技術スタック
- Frontend: Next.js App Router + TypeScript + Tailwind CSS + shadcn/ui
- BFF: Next.js Route Handler / Server Action。薄いBFFとして使う
- Backend: Laravel REST API
- DB: MySQL
- PDF: Laravel側でBlade HTML → PDF
- 認証: デモ用簡易認証、またはLaravel Sanctum

## 成功条件
1. 住民異動イベントを登録できる
2. 資格取得・変更を登録できる
3. 被保険者番号を自動付番できる
4. 被保険者詳細で資格履歴を見られる
5. 介護保険被保険者証（帳票ID: 0230010）をHTML/PDFで出力できる
6. 証発行履歴を保存・表示できる
7. 余裕があれば再交付申請・再発行まで実装する

## 非目標
- 本番品質の自治体基幹システムを作らない
- 住民記録システムとのリアル連携はしない
- 認定管理、給付管理、保険料賦課、保険料収納、滞納管理は作らない
- 住所地特例、適用除外施設、受給資格証明書、資格者証の完全実装は対象外
- BFFを厚くしない。業務ロジックはLaravelに置く
