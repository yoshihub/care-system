# 初心者向け：今回のデモで作る範囲と作らない範囲

## 今回作るものを一言でいうと

今回作るのは、

> 住民が65歳になったり、住所が変わったりしたときに、介護保険システムで被保険者として登録し、被保険者証を発行するデモ

です。

さらに余裕があれば、

> 被保険者証をなくした場合に、再交付申請を登録し、再発行するデモ

も作ります。

---

## 作る範囲：Must

Mustは、必ず作る範囲です。

| No | 作るもの | 初心者向け説明 |
|---:|---|---|
| 1 | 住民異動イベント登録 | 住民の変化をシステムに登録する |
| 2 | 住民異動イベント一覧 | 登録された住民の変化を見る |
| 3 | 資格異動登録 | 介護保険の資格取得・変更・喪失を登録する |
| 4 | 被保険者一覧 | 介護保険の対象者一覧を見る |
| 5 | 被保険者詳細 | 1人の被保険者の情報を見る |
| 6 | 資格履歴 | その人の資格の変化履歴を見る |
| 7 | 被保険者番号自動採番 | 新しく被保険者になった人に番号を付ける |
| 8 | 被保険者証HTMLプレビュー | 被保険者証の見た目を画面表示する |
| 9 | 被保険者証PDF出力 | 被保険者証をPDFで出す |
| 10 | 証発行履歴 | いつ証を発行したか履歴を残す |

---

## 作る範囲：Stretch

Stretchは、時間があれば作る範囲です。

| No | 作るもの | 初心者向け説明 |
|---:|---|---|
| 1 | 再交付申請登録 | 証をなくした・壊したなどの申請を登録する |
| 2 | 再発行処理 | 申請をもとに被保険者証を再発行する |
| 3 | 再交付申請書PDF | 再交付申請書の帳票を出す |
| 4 | 再交付履歴 | 再交付した記録を残す |

今回のデモでは、再交付はできれば作りたいですが、間に合わなければ被保険者証発行までで成立します。

---

## 作らない範囲：Out

Outは、今回絶対に作らない範囲です。

| 作らないもの | 理由 |
|---|---|
| 住所地特例 | 制度が複雑で今回のデモ範囲を超える |
| 適用除外施設 | 制度が複雑で今回のデモ範囲を超える |
| 保険料賦課 | 保険料計算は重い |
| 保険料収納 | 支払い管理は別業務 |
| 滞納管理 | 督促や差押などで業務が重い |
| 認定管理 | 要介護認定は別システム・別業務が多い |
| 給付管理 | 国保連など外部連携が重い |
| 総合事業 | 今回の被保険者資格とは別領域 |
| 住民記録システム本物連携 | PoCではCSV/手入力で代替する |
| マイナポータル連携 | 今回不要 |
| 本番用権限管理 | デモでは簡易でよい |
| 監査ログ完全対応 | 本番では必要だが今回は対象外 |
| ガバメントクラウド対応 | 今回のデモでは対象外 |

---

## 対象の業務フロー

今回の対象は次の2つです。

| 業務フロー | 内容 | 今回の扱い |
|---|---|---|
| 02-01 住民情報異動等に伴う資格異動 | 住民の変化により介護保険資格を登録・変更・喪失する | Must |
| 02-02 被保険者証等再交付 | 被保険者証をなくした場合などに再発行する | Stretch〜Must寄り |

## デモシナリオA：65歳到達による資格取得

これは一番重要なデモです。

```text
1. 65歳になる住民をCSVまたは手入力で登録する
2. 住民異動イベント一覧に表示される
3. 職員が対象者を確認する
4. 資格異動登録を実行する
5. 被保険者番号が自動で付く
6. 被保険者詳細画面に資格履歴が表示される
7. 被保険者証をプレビューする
8. 被保険者証をPDF出力する
9. 証発行履歴に記録される
```

---

## デモシナリオB：被保険者証の再交付

これは余裕があれば作るデモです。

```text
1. 既に被保険者として登録されている人を開く
2. 再交付申請を登録する
3. 申請理由を「紛失」にする
4. 再発行を実行する
5. 被保険者証が再発行される
6. 証発行履歴に「再交付」として残る
```

---

## 今回の実装で「できた」と言える条件

最低限、以下ができればデモとして成立します。

- 住民異動イベントを登録できる
- 資格異動登録できる
- 被保険者番号が付く
- 被保険者一覧に出る
- 被保険者詳細で資格履歴が見える
- 被保険者証をHTMLプレビューできる
- 被保険者証をPDF出力できる
- 証発行履歴が保存される

再交付は、間に合えば追加します。
