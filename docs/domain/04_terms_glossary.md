# 用語集

| 用語 | 説明 |
|---|---|
| 被保険者 | 介護保険の対象者 |
| 被保険者番号 | 被保険者を識別する番号 |
| 資格取得 | 介護保険の対象者になること |
| 資格喪失 | 介護保険の対象者でなくなること |
| 資格異動 | 資格取得・喪失・変更など、資格状態が変わること |
| 住民異動 | 転入、転出、死亡、住所変更、氏名変更など住民記録側の変更 |
| 被保険者証 | 介護保険の被保険者であることを示す証 |
| 再交付 | 紛失・破損などにより証を再発行すること |
| 住所地特例 | 施設入所などに伴う保険者判定の特例。今回PoCでは対象外 |
| 適用除外施設 | 介護保険の資格管理に影響する特定施設。今回PoCでは対象外 |
| BFF | Backend For Frontend。Next.jsがフロント向けAPIとしてLaravel APIを中継する構成 |
| PoC | Proof of Concept。検証用デモ |
