# 初心者向け 介護保険システム基礎

## 介護保険とは
介護が必要になった人を社会全体で支えるための制度。市区町村が保険者として運営し、対象者の資格、保険料、認定、給付などを管理する。

## 今回扱う「被保険者資格」とは
介護保険の対象者として管理するための基本情報。誰が介護保険の対象か、いつ資格を取得したか、いつ資格を失ったかを管理する。

## 被保険者とは
介護保険の対象者。今回のPoCでは主に65歳到達者を対象にする。

## 資格異動とは
資格状態が変わること。

例:
- 65歳到達 → 資格取得
- 転入 → 資格取得
- 転出 → 資格喪失
- 死亡 → 資格喪失
- 住所変更 → 資格情報変更
- 氏名変更 → 資格情報変更

## 被保険者証とは
介護保険の被保険者であることを示す証。今回のPoCではHTML/PDFで簡易的に出力する。

## 再交付とは
被保険者証をなくした、破損したなどの理由で、もう一度発行すること。

## 住民記録システムとの関係
住民の転入、転出、死亡、住所変更などは住民記録側で発生する。介護保険システムはその情報を受け取り、介護保険の資格状態を更新する。

今回のPoCでは本物の住民記録システム連携は行わない。CSV/手入力で代替する。
