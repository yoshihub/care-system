## 共通画面ルール
- 画面右上または下部に「対応業務」「機能ID」「帳票ID」を表示する
- shadcn/uiのTable, Card, Button, Badge, Dialog, Formを利用する
- 業務エラーは赤色のAlertで表示する
- pending/processed/errorなどの状態はBadgeで表示する
